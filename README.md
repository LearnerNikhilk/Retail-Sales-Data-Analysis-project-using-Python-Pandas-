# Retail Sales Data Analysis using Python (Pandas)

## Project Overview
This project focuses on analyzing a real-world retail sales dataset using Python and Pandas. The main objective of this project is to clean raw sales data, perform exploratory data analysis (EDA), and extract meaningful business insights related to product performance, customer behavior, and regional sales trends.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Work Done
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Sales analysis by category and region
- Monthly sales trend analysis
- Top-performing products identification

## Project Files
- Raw_Superstore.csv
- cleaned_superstore.csv
- SuperstoreAnalysis.ipynb

## Skills Demonstrated
- Data Cleaning using Pandas
- Data Analysis using Python
- Data Visualization
- Working with real-world datasets
